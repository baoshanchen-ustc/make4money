# Affiliate 分销系统 - UI/UX 设计文档

**版本**：v1.1
**作者**：Sally (UX Designer)
**日期**：2026-02-03
**更新记录**：v1.1 配色方案与现有系统对齐（Claude Orange 橙色调）

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **一目了然** | 核心数据（收益、进度）第一眼就能看到 |
| **激励驱动** | 用进度条、成就感设计推动用户持续分享 |
| **降低门槛** | 一键复制、一键分享，操作步骤最少化 |
| **信任透明** | 规则清晰、收益明细可查、提现流程透明 |

---

## 2. 信息架构

```
推广中心（入口）
├── 首页 Dashboard
│   ├── 收益卡片（累计/可提现）
│   ├── 推广数据（邀请数/有效数）
│   ├── 阶梯进度条
│   └── 快捷操作区
│
├── 邀请工具
│   ├── 复制链接
│   ├── 生成海报（3套模板）
│   └── 二维码下载
│
├── 邀请记录
│   └── 列表（筛选：全部/有效/待激活）
│
├── 佣金明细
│   └── 列表（筛选：全部/待确认/已确认）
│
├── 提现
│   ├── 提现申请（微信/支付宝）
│   └── 提现记录
│
└── 规则说明
    └── FAQ + 规则详情
```
